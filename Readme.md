## Supported iOS Firmware

- **iOS 15 - 16.3**

## Required Installations

1. **Install Python**
   - Download Python from [python.org](https://www.python.org/downloads/).

2. **Install libimobiledevice**

   ```
   brew install libimobiledevice
   ```
   ```
   brew install usbmuxd
   ```
